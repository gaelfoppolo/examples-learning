Test d'inclusion

Attributs

  • intervalle : incluse dans celle de la solution
    ex : [2;4] est inclu dans [2;6]
  • liste : incluse dans celle de la solution
    ex : (rouge, jaune) inclu dans (vert, jaune, rouge)
  • arbre : est un des descendants de la solution
    ex : carré est un descendant de polygone

Relations : possède exactement les mêmes relations
