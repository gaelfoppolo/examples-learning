Calculation of the index of the combination in the array

\(n\): number of examples - 1
\(O_i\): index of the object in the example \(i\)
\(E_j\): number of objects in the example \(j\)

\[ \left( \sum\limits_{i = 0}^{n-1} \left( O_i \times \prod\limits_{j=i+1}^{n} E_j \right) \right) +O_n \]
