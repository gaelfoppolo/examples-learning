Options

Syntaxe Action
--help | -h Affiche l'aide pour le lancement et les options disponibles
--expand-relations Si la solution a des relations, affiche les objets liés plutôt que leur nom
--no-generalization Saute l'étape de généralisation (affiche toutes les solutions, même les doublons)
--no-counter-examples Saute l'étape des contre-exemples. Évite la suppression de solutions
--no-color Supprime toutes les couleurs de l'affichage (utile pour rediriger la sortie dans un fichier par exemple)
-v Niveau de verbosité, variant de 1 à 4
